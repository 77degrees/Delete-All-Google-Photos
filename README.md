# Delete All Google Photos Chrome Extension

A Chrome extension that automates bulk deletion of your Google Photos since Google doesn't allow you to delete all photos at once.

## Features

- **Bulk Delete**: Automatically selects and deletes multiple photos at once
- **Easy to Use**: Simple one-click interface
- **Works with Google Photos**: Integrates directly with Google Photos web interface

## Installation

### Manual Installation (Developer Mode)
1. Download or clone this repository
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode" in the top right corner
4. Click "Load unpacked"
5. Select the `extension/2.4_0` folder from this repository

## Usage

1. Click on the extension icon in your Chrome toolbar
2. Click the "Delete all photos" button
3. A new tab will open with Google Photos asking you to confirm deletion
4. Follow the on-screen prompts to complete the deletion process

## Important Notes

- **Backup First**: Before using this extension, make sure you have backed up any photos you want to keep
- **Irreversible Action**: Deleted photos will be moved to trash and permanently deleted after 60 days
- **Google Photos Only**: This extension only works with Google Photos (photos.google.com)

## Permissions

This extension requires the following permissions:
- **Storage**: To save user preferences and settings
